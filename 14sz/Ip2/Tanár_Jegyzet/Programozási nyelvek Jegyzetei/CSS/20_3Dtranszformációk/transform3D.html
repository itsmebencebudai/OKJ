Elmozdítás(translate)

Elmozdítás X irányban:
transform: translateX(30px);
Elmozdítás Y irányban:
transform: translateY(30px);
Elmozdítás Z irányban:
transform: translateZ(300px);
Elmozdítás 3D-ben:
transform: translate3d(30px,40px,10px);;

Forgatás(rotate)

Forgatás X tengelyen:
transform: rotateX(60deg);
Forgatás Y tengelyen:
transform: rotateY(60deg);
Forgatás Z tengelyes:
transform: rotateZ(30deg);
Forgatás 3D-ben(X,Y,Z tengelyen):
transform: rotate3d(1, 1, 1, 45deg);

Átméretezés(scale)

Átméretezés X irányban:
transform: scaleX(1.5);
Átméretezés Y irányban:
transform: scaleY(0.5);
Átméretezés Z irányban:
transform: scaleZ(2);
Átméretezés Y irányban:
transform: scale3d(0.5, 2, 2);

Minden egyben(matrix)

3D ALL IN ONE:
transform: matrix3d(1, 0.2, 0.8, 0, -0.5, 0.8, 0.2, 0, -0.6, -0.5, 0.5, 0, 0, 0, 0, 1);
