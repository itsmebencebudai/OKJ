Alap animáció

Weboldal betöltődésével indul:

.alapAnimacio{
animation-name:ketAllapotosSzinezo;
animation-duration:4s;
}
Akkor indul, ha fölé megyünk egérrel:

.alapAnimacio2:hover{
animation-name:negyAllapotosSzinezo;
animation-duration:10s;
}

Késleltetett animáció

2 másodperccel azután indul, hogy fölé mentünk egérrel:

.kesleltetettAnimacio:hover{
animation-name:ketAllapotosSzinezo;
animation-duration:4s;
animation-delay: 2s;
}

Ismétlésszám megadása

Háromszor ismétlődik meg, a weboldal betöltődése után 1 másodperccel

.ismetlodoAnimacio{
animation-name:mozgato;
animation-duration:4s;
animation-delay:1s;
animation-iteration-count: 3;
}
VÉGTELENSZER ismétlődik meg, a weboldal betöltődése után 2 másodperccel

.ismetlodoAnimacio{
animation-name:mozgato;
animation-duration:4s;
animation-delay:2s;
animation-iteration-count: infinite;
}
